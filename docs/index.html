Introduction to the BlueMicro_BLE Firmware

Getting Started: in 4 simple steps

1: Install Arduino IDE

Watch the video above to see how to install the Arduino IDE and get the library needed.

How to install tools
2: Download BlueMicro_BLE Firmware

To get a copy of the firmware, you have 3 options: (Video will be updated)

  • Download the zip file
  • Clone the repository localy
    git clone https://github.com/jpconstantineau/BlueMicro_BLE.git
  • Fork the repo and download/clone locally

Simplest is just to download the zip file.

Download directly from github
3: Edit 4 Keyboard Files

Open the firmware in the IDE, edit the following 4 files: (Video will be updated)

  • hardware_config.h: for the GPIO definition of your keyboard
  • keyboard_config.h: for the BLE definition of your keyboard
  • keymap.cpp: for the keymaps definitions
  • keymap.h: to support the keymaps definitions

Examples of these files for different keyboards are in the keyboards folder

How to configure Keymaps
4: Compile and Upload

(Video will be updated)

  • Select appropriate "Board" in the "Tools" menu
  • Compile
  • Connect your keyboard with USB cable
  • Select apropriate serial "Port" in the "Tools" menu
  • Upload - Depending on your hardware/firmware, you may need to double-press reset to put it in DFU mode
  • Go to your computer's Bluetooth connections and "add device"
  • You should be able to pair your keyboard
